# فروشگاه کتاب آنلاین (Online Bookstore)

فروشگاه کتاب آنلاین (Online Bookstore) یک اپلیکیشن وب فول‌استک و مدرن است که با استفاده از پایتون و فریم‌ورک فلسک (Flask) توسعه یافته است. این پروژه با بهره‌گیری از معماری سه‌لایه (سرویس‌گرا)، تفکیک کامل منطق تجاری (Service Layer) و پیاده‌سازی ماژولار (Blueprints)، ساختاری مقیاس‌پذیر و استاندارد را برای یک سیستم فروشگاهی آنلاین ارائه می‌دهد.

## 🌟 ویژگی‌های اصلی

### بخش مشتریان (Frontend)
- **سیستم احراز هویت امن:** ثبت‌نام، ورود و خروج کاربران با استفاده از `Flask-Login` و هش امن رمزهای عبور.
- **پنل کاربری (پروفایل):** امکان مشاهده و ویرایش اطلاعات حساب کاربری.
- **دفترچه آدرس پویا:** امکان ثبت چندین آدرس، حذف آدرس‌ها و انتخاب یکی از آدرس‌ها به عنوان آدرس «پیش‌فرض» برای تحویل کالا.
- **تاریخچه سفارشات:** مشاهده لیست سفارش‌های قبلی به همراه جزئیات کامل فاکتور و نام کتاب‌های خریداری‌شده.
- **سبد خرید تعاملی:** افزودن، حذف و به‌روزرسانی تعداد محصولات با استفاده از `session` در مرورگر (پاکسازی خودکار سشن سبد خرید در زمان خروج).
- **تسویه حساب هوشمند:** پر شدن خودکار فرم اطلاعات ارسال بر اساس آدرس پیش‌فرض کاربر در صفحه تسویه حساب (Checkout) با قابلیت ویرایش لحظه‌ای.

### پنل مدیریت (Backend)
- **امنیت سراسری پنل ادمین:** محافظت از تمام روت‌های ادمین به صورت یکپارچه با استفاده از هوک‌های سراسری فلاسک (`before_request`) و سیستم نقش‌بری.
- **ورود یکپارچه (Unified Login):** سیستم لاگین مشترک که به صورت خودکار کاربران ادمین را به داشبورد و کاربران عادی را به فروشگاه هدایت می‌کند.
- **داشبورد آماری:** نمایش شاخص‌های کلیدی مانند مجموع درآمد، سفارشات معلق، سفارشات ارسال‌شده و تعداد کل کاربران ثبت‌نام شده.
- **مدیریت کاربران (User Management):** مشاهده لیست مشتریان، تعداد آدرس‌ها و تعداد کل خریدهای هر کاربر.
- **مدیریت نقش‌ها:** امکان ارتقای کاربران به مدیر سیستم (Admin) یا تنزل آن‌ها به مشتری عادی به صورت مستقیم از پنل.
- **مدیریت کامل محصولات (CRUD):** ایجاد، ویرایش و حذف کتاب‌ها به همراه آپلود و جایگزینی هوشمند تصاویر جلد (حفظ تصویر قبلی در صورت عدم آپلود تصویر جدید).
- **مدیریت سفارشات:** مشاهده جزئیات فاکتور مشتریان و امکان تغییر وضعیت سفارش‌ها به "ارسال شده".

## 🛠️ تکنولوژی‌های استفاده شده (Tech Stack)

- **بک‌اند:**
  - **زبان:** Python 3
  - **فریم‌ورک:** Flask
  - **دیتابیس:** SQLite
  - **مدیریت مهاجرت دیتابیس:** Flask-Migrate (پیکربندی شده در حالت Batch Mode برای سازگاری کامل با محدودیت‌های SQLite)
  - **ORM:** SQLAlchemy (Flask-SQLAlchemy)
  - **احراز هویت:** Flask-Login
- **فرانت‌اند:**
  - **قالب‌سازی:** Jinja2
  - **طراحی و استایل:** HTML5, CSS3, Bootstrap 5 (قالب NiceShop)
- **معماری:**
  - استفاده از Blueprints برای ماژول‌بندی روت‌ها
  - پیاده‌سازی الگوهای طراحی Service Layer برای تفکیک منطق تجاری دیتابیس از کنترلرها

## 🚀 راهنمای راه‌اندازی و اجرا

برای اجرای این پروژه به صورت محلی، مراحل زیر را دنبال کنید:

1.  **کلون کردن ریپازیتوری:**
    ```bash
    git clone https://github.com/minadarvishi/bookstore.git
    cd bookstore
    ```

2.  **ساخت و فعال‌سازی محیط مجازی:**
    ```bash
    python -m venv .venv
    # Windows
    source .venv/Scripts/activate
    # macOS/Linux
    source .venv/bin/activate
    ```

3.  **نصب نیازمندی‌ها:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **اعمال ساختار دیتابیس (Migrations):**
    از آنجا که تغییرات دیتابیس با Flask-Migrate ثبت شده‌اند، با دستور زیر دیتابیس و جداول را بسازید:
    ```bash
    flask db upgrade
    ```

5.  **اجرای برنامه:**
    ```bash
    flask run
    ```
    حالا اپلیکیشن در آدرس `http://127.0.0.1:5000` در دسترس است.

6.  **دسترسی به پنل ادمین:**
   اگر پروژه را روی سیستم خودتان بالا آورده‌اید، ابتدا یک حساب کاربری عادی در سایت بسازید. سپس برای تبدیل آن به ادمین، دستورات زیر را در ترمینال بزنید:
    ```bash
    flask shell
    >>> from models import User, db
    >>> admin = User.query.filter_by(email="your-email@example.com").first()
    >>> admin.is_admin = True
    >>> db.session.commit()
    >>> exit()
    ```
